# AetherAI

**マルチモデル デスクトップAIチャットクライアント · Electron + React**

[English](./README.en.md) · [简体中文](./README.md) · [日本語](./README.ja.md)

</div>

AetherAI は、複数の LLM プロバイダ（OpenAI / Claude / DeepSeek / ローカルモデルなど）を一つのインターフェースに統合する、ローカル優先のデスクトップ AI チャットアプリです。マルチセッション同時ストリーミング、アリーナ比較、ペルソナプリセット、ファイル・画像添付、ツール呼び出し（Web検索 / ファイル読み取り）、カスタム背景、思考レベル制御などに対応します。

## ✨ 機能

- **マルチプロバイダ管理** — プロバイダ/モデル アダプタレイヤー、プロバイダ追加は1ファイル
- **マルチセッション同時ストリーミング** — 一方のセッションが生成中でも別セッションで継続可能
- **アリーナ** — 同一質問に複数モデルが同時回答、ベストに投票、ELO レーティング自動更新
- **ペルソナ** — システムプロンプトのプリセット、セッション単位で切替
- **添付ファイル** — テキストはコンテキスト注入、画像はマルチモーダル（対応モデルが必要）
- **長文ペースト折りたたみ** — 数百行のペーストを展開可能なスニペットに自動圧縮（ChatGPT風）
- **ツール呼び出し（Agent）** — 内蔵 `read_file` / `web_search`、折りたたみ可能なツール呼び出しブロック
- **思考レベルスライダー** — 実効果: OpenAI o系 → `reasoning_effort`、Claude → `thinking.budget_tokens`
- **スマートサイドバー要約** — タイトルはモデル生成のトピックフレーズ（原文コピーではない）
- **カスタム背景** — 画像アップロード + 透明度 / ぼかし調整
- **テーマ** — ライト / ダーク / ブルー / グラス / レトロ
- **多言語** — 中文 / English
- **ローカル保存** — 全データはローカル SQLite、何もアップロードしない

## 🚀 クイックスタート

### 前提
- Node.js 18+
- npm 9+

### インストールと実行
```bash
cd app
npm install
npm run dev      # 開発モード（ホットリロード）
npm run build    # 本番フロントエンドをビルド
npm start        # Electron 起動
```

またはプロジェクトルートの `start.bat`（Windows）を実行。

### 最初のプロバイダ設定
1. 起動後、サイドバーの「モデル管理」をクリック
2. プロバイダを追加（名前 / API URL / API Key）
3. 「モデル取得」で利用可能なモデル一覧を取得
4. チャットページに戻って会話開始

## 🔒 プライバシー

**全データはローカルに保存されます。** AetherAI はユーザーデータを収集・アップロードしません。API キー、チャット履歴、ペルソナはすべてローカルの SQLite データベースに保存されます。唯一の外部通信は、設定した LLM プロバイダ API へのリクエストです。

> ⚠️ GitHub にプッシュする前に、`.gitignore` が `*.db`、`dist/`、`node_modules/` などを除外していることを確認してください。
